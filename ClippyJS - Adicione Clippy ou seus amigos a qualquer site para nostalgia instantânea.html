Clippy.js

Adicione Clippy ou seus amigos a qualquer site para nostalgia instantânea. Nossa pesquisa mostra que as pessoas adoram duas coisas: tecnologias Microsoft com falha e bibliotecas Javascript obscuras. Naturalmente, decidimos combinar os dois.

Demonstração

Clippy.js é uma implementação Javascript completa do Microsoft Agent (AKA Clippy e amigos), pronta para ser incorporada em qualquer site. Escolha um assistente abaixo e amasse alguns botões de animação! O nosso favorito é o Links the cat.

Motivação

Durante o almoço, algumas semanas atrás, pensamos que seria legal ter um Clippy totalmente funcional que possa ser incorporado em qualquer site. Nós brincamos com isso e nos divertimos muito, rindo de Clippy e de suas travessuras.

We started thinking about the developers' state of mind when they created Clippy. Did they think it would really help people? It seems that Microsoft really believed that Assistants were the way of the future.

We built Clippy.js over the weekend to share that fun and whimsy with everyone, and to remind people to try new and risky things, even when they seem silly.

Features

  • Play agent animations
  • "Speak" to the user
  • Move the agent around
  • Use multiple agents at the same time

Behind the scenes

Agents are composed of multiple sets of animations. However, unlike your Saturday morning cartoon, the animation is not linear. For example, while Links the cat is idle, behind the scenes a coin is flipped, and 10% of the time Links will turn about in boredom, or scratch behind his ears.

Usage: Setup

Add this code to you to your page to enable Clippy.js. 

<!-- Add the stylesheet to the head -->
<link rel="stylesheet" type="text/css" href="clippy.css" media="all">

...

<!-- Add these scripts to  the bottom of the page -->
<!-- jQuery 1.7+ -->
<script src="jquery.1.7.min.js"></script>

<!-- Clippy.js -->
<script src="clippy.min.js"></script>

<!-- Init script -->
<script type="text/javascript">
    clippy.load('Merlin', function(agent) {
        // Do anything with the loaded agent
        agent.show();
    });
</script>

Usage: Actions

All the agent actions are queued and executed by order, so you could stack them.

Corre 
// Play a given animation
agent.Play('Searching');
Corre 
// Play a random animation
agent.animate();
// Get a list of all the animations
agent.animations();
// => ["MoveLeft", "Congratulate", "Hide", "Pleased", "Acknowledge", ...]
Corre 
// Show text balloon
agent.speak('When all else fails, bind some paper together. My name is Clippy.');
Corre 
// Move to the given point, use animation if available
agent.moveTo(100,100);
Corre 
// Gesture at a given point (if gesture animation is available)
agent.gestureAt(200,200);
Corre 
// Stop the current action in the queue
agent.stopCurrent();
Corre 
// Stop all actions in the queue and go back to idle mode
agent.stop();
